# claude-marketplace

Personal Claude Code plugin marketplace for [@awaisali88](https://github.com/awaisali88).

## Install

Register the marketplace once per device:

```
claude plugin marketplace add awaisali88/claude-marketplace
```

Then install plugins:

```
claude plugin install aria-trading@claude-marketplace
```

## Plugins

| Name | Description | Source |
| --- | --- | --- |
| `aria-trading` | Aria trading skill | [awaisali88/aria-trading-skill](https://github.com/awaisali88/aria-trading-skill) |
